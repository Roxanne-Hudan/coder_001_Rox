# Django

## Crear un proyecto

Clonar un repositorio 

Crear un entorno virtual 

Instalar Dependencias

Crear un proyecto: 

    mkdir src
    cd src
    django-admin startproject config .

Ejecutar el servidor:

    python manage.py runserver

Cambiar el Idioma en src/config/settings.py
